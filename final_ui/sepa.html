Impact Analysis

GENERAL CONSIDERATIONS

  • Migration to the 2019 message version of the ISO 20022 standard.
  • Identification/Organisation Identification: SEPA Usage Rule updated to allow either 'AnyBIC or BEI', 'LEI' or one occurrence of 'Other'.
  • Possibility to provide a structured address of the payment end-user: Changes to 'Debtor/Postal Address' and 'Creditor/Postal Address' including sub-elements being shaded yellow and new SEPA Usage Rules for both:
    • If 'Address Line' is used, then sub-elements other than 'Country' are forbidden.
    • A combination of 'Address Line' and 'Country' is allowed (with recommendations depending on EEA/non-EEA location of Debtor/Creditor Agents).
    • If 'Address Line' is not used, then at least 'Town Name' and 'Country' must be used.
  • 'Debtor Account' and 'Creditor Account': New optional sub-element 'Proxy' added with associated attributes.
  • 'Assigner/Assignee': 'LEI' can be used in addition to 'Name' to indicate a CSM when it has no BIC.
  • 'Remittance Information/Unstructured': New SEPA Usage Rule added regarding providing commercial trade name, location, date and time if different from legal name and address.
